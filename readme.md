# BPJS Fraud Detection System

Production-ready fraud detection system with SHAP explainability and RAG-LLM integration support.

## Features

- ✅ **Two-stage detection**: Binary (fraud vs benign) → Multiclass (fraud types)
- ✅ **Ensemble models**: XGBoost + LightGBM + RandomForest
- ✅ **SHAP explanations**: Local interpretability for each prediction
- ✅ **Threshold optimization**: Automatic threshold tuning for optimal F1/precision/recall
- ✅ **Modular architecture**: Clean separation of concerns
- ✅ **RAG-ready output**: Merged predictions + original data for LLM integration
- ✅ **Production inference**: Efficient batch and single prediction
